# 基于改进SimCLRv2的自监督小样本农作物病害识别系统

## 项目简介

本项目为《深度学习》课程设计作业，实现了基于自监督对比学习的农作物病害识别系统，在仅使用5%标注数据的情况下达到92.7%的识别准确率。

## 环境依赖

- Python 3.11+
- PyTorch
- torchvision
- numpy,matplotlib,scikit-learn,seaborn,tqdm

## 数据集获取

由于github仓库对上传的文件大小有限制，因此本次作业课程设计只为老师展示出了最终的测试成果以及测试图集，数据测试集的体量过大，这里为老师提供数据集的下载链接，老师下载后可自行在终端上进行测试并验证测试结果。

本项目使用 PlantVillage 公开数据集，下载方式：

1. Kaggle 地址：https://www.kaggle.com/datasets/emmarex/plantdisease 
2. 下载解压后将数据集放入 `PlantVillage` 文件夹（项目根目录下）即可运行

## 运行说明

1. 安装依赖：`pip install torch torchvision numpy matplotlib scikit-learn seaborn tqdm`
2. 下载数据集并放置到对应路径 
3. 运行 `深度学习课程设计报告.ipynb` 即可完成训练与评估
